## Popis projektu

Projekt jsem stavěl v roce 2023, cílem bylo postavit informační web. Bylo to poprvé co jsem zkoušel stavět web s informační web NextJS. Už jsem měl základy
s React knihovnou a uměl pracovat v framework protože jsem už používal alternativu Sveltekit.

Barevné schéma bylo Lime barva tailwindcss, v praxi to byla světle zelená barva, jako limetka. Web má blog který se ovládá přes sanity CMS.
Integrace sanity do aplikace byla nejtěžší část aplikace. Díky sanity mohou i normální uživatelé psát do blogu aniž by museli editovat kód a ani nic měnit.

Stránka byla psána v angličtině, což je většina mých projektů.
